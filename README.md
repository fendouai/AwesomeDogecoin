# 狗币，狗狗币，DogeCoin

Dogecoin，有人称作“狗狗币/狗币”，诞生于2013年12月8日，基于Scrypt算法，是国际上用户数仅次于比特币的第二大虚拟货币。

## 发展历史

*  2013年12月8日诞生。
*  2017年6月，突破 0.003美元。
*  2018年1月，突破 0.02美元。
*  2020年1月，上涨超过 500%，突破 0.08美元。
